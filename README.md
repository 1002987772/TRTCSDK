## 最新版本 6.7.7733 @ 2019.09.29

1. 【iOS】新增耳返支持。
2. 【Android】AAR 打包新增权限获取配置。
3. 【Android】增加 Android 8.0以上系统 CPU 占用评估。
4. 【windows】新增音效接口支持。
5. 【windows】新增64位 C# API 支持。
6. 【MAC】优化进房和出帧速度。
7. 【全平台】转推耗时优化。
8. 【全平台】支持单个用户播放音量独立调节能力。
